Variables

State variables

  • PlasticSlip:
    • variable name: g
    • variable type: strain
    • array size: 12
    • description: plastic slip
  • ElasticPartOfTheDeformationGradient:
    • variable name: Fe
    • variable type: DeformationGradientTensor
    • description: elastic part of the deformation gradient
  • EquivalentViscoplasticSlip:
    • variable name: p
    • variable type: strain
    • array size: 12
  • BackStrain:
    • variable name: a
    • variable type: strain
    • array size: 12

External state variables

  • Temperature:
    • variable name: T
    • variable type: temperature
    • description: the temperature

Parameters

  • epsilon:
    • variable type: NumericType
    • description: value used to stop the iteration of the implicit algorithm
    • default value for Tridimensional: 1e-14
  • theta:
    • variable type: NumericType
    • description: theta value used by the implicit scheme
    • default value for Tridimensional: 1
  • YoungModulus1:
    • variable name: young1
    • variable type: stress
    • description: the Young’s modulus of an isotropic material along the first direction of orthotropy
    • default value for Tridimensional: 208000
  • YoungModulus2:
    • variable name: young2
    • variable type: stress
    • description: the Young’s modulus of an isotropic material along the second direction of orthotropy
    • default value for Tridimensional: 208000
  • YoungModulus3:
    • variable name: young3
    • variable type: stress
    • description: the Young’s modulus of an isotropic material along the third direction of orthotropy
    • default value for Tridimensional: 208000
  • PoissonRatio12:
    • variable name: nu12
    • variable type: real
    • description: the Poisson’s coefficient between the first and second directions of orthotropy
    • default value for Tridimensional: 0.3
  • PoissonRatio23:
    • variable name: nu23
    • variable type: real
    • description: the Poisson’s coefficient between the second and third directions of orthotropy
    • default value for Tridimensional: 0.3
  • PoissonRatio13:
    • variable name: nu13
    • variable type: real
    • description: the Poisson’s coefficient between the first and third directions of orthotropy
    • default value for Tridimensional: 0.3
  • ShearModulus12:
    • variable name: mu12
    • variable type: stress
    • description: the shear moduls between the first and second directions of orthotropy
    • default value for Tridimensional: 80000
  • ShearModulus23:
    • variable name: mu23
    • variable type: stress
    • description: the shear moduls between the second and third directions of orthotropy
    • default value for Tridimensional: 80000
  • ShearModulus13:
    • variable name: mu13
    • variable type: stress
    • description: the shear moduls between the first and third directions of orthotropy
    • default value for Tridimensional: 80000
  • n:
    • variable type: real
    • default value for Tridimensional: 10
  • K:
    • variable type: real
    • default value for Tridimensional: 25
  • tau0:
    • variable type: real
    • default value for Tridimensional: 66.62
  • Q:
    • variable type: real
    • default value for Tridimensional: 11.43
  • b:
    • variable type: real
    • default value for Tridimensional: 2.1
  • d:
    • variable type: real
    • default value for Tridimensional: 494
  • C:
    • variable type: real
    • default value for Tridimensional: 14363
  • minimal_time_step_scaling_factor:
    • variable type: real
    • description: minimal value for the time step scaling factor
    • default value for Tridimensional: 0.1
  • maximal_time_step_scaling_factor:
    • variable type: real
    • description: maximal value for the time step scaling factor
    • default value for Tridimensional: 1.79769e+308
  • numerical_jacobian_epsilon:
    • variable type: NumericType
    • description: perturbation value used to compute a numerical approximation of the jacobian
    • default value for Tridimensional: 1e-15
  • iterMax:
    • variable type: ushort
    • description: maximum number of iterations allowed
    • default value for Tridimensional: 100

    • variable type: StiffnessTensor
    • description: stiffness tensor computed from elastic material properties

    • variable type: StiffnessTensor&
