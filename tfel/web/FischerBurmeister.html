On the Fisher-Burmeister complementary function and its applications to multi-surface plasticity

Thomas Helfer

17/01/2019

  • The Fisher-Burmeister complementary function
    • Link with Karush-Kuhn-Tucker condition of plasticity
    • A regularized version of the Fisher-Burmeister complementary function
  • Acknowledgements

The Fisher-Burmeister complementary function

The Fisher-Burmeister complementary function \(\Phi{\left(x,y\right)}\) is defined as follows:

\[ \Phi{\left(x,y\right)}=x+y-\sqrt{x^{2}+y^{2}} \]

\(\Phi{\left(x,y\right)}\) has the following property:

\[ \Phi{\left(x,y\right)}=0 \quad\Leftrightarrow\quad \left\{ \begin{aligned} x&\geq 0\\ y&\geq 0\\ x\,y&=0 \end{aligned} \right. \qquad(1)\]

Link with Karush-Kuhn-Tucker condition of plasticity

Equation (1) clearly reflects the classical Karush-Kuhn-Tucker condition of plasticity:

\[ \left\{ \begin{aligned} \dot{\lambda}&\geq 0\\ f&\leq 0\\ \dot{\lambda}\,f&=0 \end{aligned} \right. \qquad(2)\]

where:

  • \(\dot{\lambda}\) is the plastic multiplier.
  • \(f\) is a the yield function such that:
    • \(f< 0\) defines the elastic domain.
    • \(f= 0\) defines the yield surface.

In other words, finding a plastic multiplier \(\dot{\lambda}\) satisfying Conditions (2) is equivalent to find the root of the following equation:

\[ \Phi{\left(\dot{\lambda},-f\right)}=0 \qquad(3)\]

If several plastic surfaces are involved, each plastic multipliers must satisfy an equation similar to (3). In an implicit formulation, this remove the need of using a status method to determine which mechanisms are active or inactive.

A regularized version of the Fisher-Burmeister complementary function

\(\Phi{\left(x,y\right)}\) is differentiable everywhere, except at \(0,0\). For this reason, following [1], we introduce a regularized version denoted \(\Phi_{\varepsilon}{\left(x,y\right)}\):

\[ \Phi_{\varepsilon}{\left(x,y\right)}=x+y-\sqrt{x^{2}+y^{2}+\varepsilon^{2}} \]

where \(\varepsilon\) is a small regularization parameter. \(\Phi_{\varepsilon}{\left(x,y\right)}\) has the following property:

\[ \Phi_{\varepsilon}{\left(x,y\right)}=0 \quad\Leftrightarrow\quad \left\{ \begin{aligned} x&\geq 0\\ y&\geq 0\\ 2\,x\,y&=\varepsilon^{2} \end{aligned} \right. \qquad(4)\]

Acknowledgements

The authors are grateful to Y. Chemisky, main developer of the Simcoon library, for introducing them to the use of the Fisher-Burmeister complementary function in multi-surfaces plasticity.

1.
Liao-McPherson, Dominic, Huang, Mike and Kolmanovsky, Ilya. A regularized and smoothed fischer–burmeister method for quadratic programming with applications to model predictive control. IEEE Transactions on Automatic Control. July 2019. Vol. 64, no. 7, p. 2937–2944. DOI 10.1109/TAC.2018.2872201.
