Detailed Description

Contains basic mathematical functions and routines.

Note that:

  • Some functions are declared as INLINE functions for maximum efficiency. As a consequence, they are declared AND defined in the .hxx file.
  • Some functions have their BLAS or LAPACK equivalent, which are specifically designed to hanlde very large objects and are not efficient for small calculations. As a consequence, you should prefer the following functions for 3D calculations.
