template<typename T , typename List >
T::return_type tfel::utilities::apply ( const GenTypeBase< List > &  v)

Apply function T::apply to a GenTypeBase for the type holded by the GenTypeBase.

Parameters
T: class containing the function to be applied.
List: list of types contained in the GenTypeBase class.
v: GenTypeBase class argument.
Precondition
class T must contain a static function name 'apply' for each type in List.
class T must define 'return_type' as a class or a typedef.
Exceptions
aGenTypeCastError if the GenTypeBase does not hold any object.
