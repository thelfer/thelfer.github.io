Detailed Description

an abstract class for models

Most models are built on balance equation (heat transfer, mechanics, diffusion, etc..) which requires to solve partial differential equations (PDEs).

Models may also defines internal state variables which are generally defined at integration points.

Unknown fields and internal state variables are generally sufficient to define the state of the model.
