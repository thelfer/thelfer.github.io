inlinestatic

It returns the first iterator i in [first, first+N) such that no other iterator in [first, last) points to a value greater than *i. The return value is first+N if and only if [first, last) is an empty range.

Parameters
ForwardIterator,iteratorto the elements.
Returns
ForwardIterator to the minimum element
Precondition
  • ForwardIterator is a model of Forward Iterator.
  • ForwardIterator's value type is LessThan Comparable.
