Detailed Description

BehaviourBrick describing standard elasticity for small strain behaviours and handles various enhancements to behaviours based on standard elasticity:

  • Generic support to plane stress.
  • Generic way to compute the tangent operator (implicit parser only) This extra features can be disabled using dedicated parameters.

This BehaviourBricks allows the followig parameters:

  • Isotropic. This parameter can be used to declare that the elastic stiffness is isotropic even though the behaviour is orthotropic. This parameter can not be used if the @RequiresStiffnessTensor has been used.
  • Orthotropic. This requires the behaviour to de declared orthotropic (through @OrthotropicBehaviour keyword). This parameter can not be used if the @RequiresStiffnessTensor has been used.

This behaviour brick relies on the Hooke stress potential. The options passed to this brick are forwarded to the Hooke stress potential.
