◆ registerDefaultCallBacks()

virtual void tfel::utilities::ArgumentParser::registerDefaultCallBacks ( )
protectedvirtual

register a default callbacks

The default callbacks are :

  • '–help', '-h' which displays the list of the command line arguments (if the treatHelp method is not overriden if the derivate class)
  • '–version', '-v' which displays the current code version using the getVersionDescription() method of the derivate class (if the 'treatVersion' method is not overriden if the derivate class)
