Detailed Description

template<unsigned int N>
class tfel::fsalgo::accumulate< N >

accumulate computes the sum (or some other binary operation) of N elements of a container.

Parameters
Nnumber of element to be summed.

This documentation is mostly inspired from: http://www.sgi.com/tech/stl/transform.html

result of the summation:

\[\displaystyle\mathrm{init}+\sum_{i=0}^{N-1}*(p+i)\]

Precondition
  • InputIterator is a model of Input Iterator.
  • T is a model of Assignable.
  • If x is an object of type T and y is an object of InputIterator's value type, then x + y is defined.
  • The return type of x + y is convertible to T.

  • The value type of InputIterator is convertible to BinaryFunction's second argument type.
  • BinaryFunction's return type is convertible to T.
