Detailed Description

template<std::size_t N, typename CharT, typename Traits>
struct tfel::utilities::basic_fcstring< N, CharT, Traits >

This class implements a string with fixed capacity. In pratice, a simple wrapper around std::array<CharT> providing an API inspired by std::basic_string.

Template Parameters
CharTType of character
TraitsTraits for character type, defaults to std::char_traits<CharT>.
