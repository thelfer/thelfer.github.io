Detailed Description

template<unsigned short N, typename NumericType, typename Child, template< unsigned short, typename > typename ExternalWorkSpace = StackAllocatedTinyLevenbergMarquardtSolverWorkspace>
struct tfel::math::TinyLevenbergMarquardtSolver< N, NumericType, Child, ExternalWorkSpace >

A class based on the curiously recurring template pattern (CRTP) to solve system of non linear equations using the Newton-Raphson algorithm.

Template Parameters
Nsize of the system of non linear equations.
NumericTypenumeric type.
Childbase class.
ExternalWorkSpaceclass containing data members used by the solver.

By default, the Child class must:

  1. Set the values of epsilon and iterMax data members and the initial guess of the unknowns, i.e. by setting the value of the zeros data member.
  2. Provide a method called computeResidual which must computes the residual, i.e. the data member fzeros and the jacobian matrix, i.e. the data member jacobian using the current estimate of the solution, i.e. the data member zeros.
