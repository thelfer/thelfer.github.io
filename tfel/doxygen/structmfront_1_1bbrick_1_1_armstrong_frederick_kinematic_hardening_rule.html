Detailed Description

class describing the Armstrong-Frederick kinematic hardening rule defined by the following evolution of the back-strain variable \(\underline{a}\):

\[ \underline{\dot{a}}=\underline{\dot{\varepsilon}}^{p}-D\,\underline{a} \]
