Detailed Description

template<typename ResultType, typename Operation>
struct tfel::math::Expr< ResultType, Operation >

an Expr object allows the lazy evaluation of a mathematical operation.

Template Parameters
ResultType: the type of the result of the operation
Operation: Operation to be performed

Partial specialisation are provided for:

  • unary operations (negation, function call)
  • binary operations
