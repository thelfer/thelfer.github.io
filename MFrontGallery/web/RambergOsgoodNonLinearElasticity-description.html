Description of behaviour RambergOsgoodNonLinearElasticity

L . Gelebart / V . Duc Nguyen

  • List of supported Hypotheses
  • Variables
    • Material properties
    • External state variables
    • Parameters

Ramberg-Osgood model (non-(linear elastic formulation) Eps = S:Sig +β.σ_eq^n. N with: - N = (3/2) Sig_dev / σ_eq - σ_eq = rac((3/2)σ_dev:σ_dev)) - β = α⋅σ₀/E

  • Axisymmetrical
  • PlaneStrain
  • GeneralisedPlaneStrain
  • Tridimensional
