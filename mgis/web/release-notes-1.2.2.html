MFrontGenericInterfaceSupport Version 1.2.2

Thomas Helfer, Jérémy Bleyer

2020

  • 1 New features
    • 1.1 Add support for point-wise models
  • 2 Issues solved
    • 2.1 Issue #66: Add support for point-wise models

This version is compatible with behaviours generated by Version 3.4.3 of the TFEL/MFront project using the generic interface. This version was released on October, 28 2021.

1 New features

1.1 Add support for point-wise models

Point-wise models, i.e. behaviours without gradient, thermodynamic force nor tangent operator block, are now supported.

The c, fortran and python bindings have updated accordingly.

2 Issues solved

2.1 Issue #66: Add support for point-wise models

For more details, see https://github.com/thelfer/MFrontGenericInterfaceSupport/issues/73
