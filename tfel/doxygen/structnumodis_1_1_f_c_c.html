Detailed Description

FCC crystallography handler using the Wigner-Seitz cell.

FCC handles Face-Centered Cubic crystallographic structures using the Wigner-Seitz cell.

The unit vectors \(\mathbf{a}_1\), \(\mathbf{a}_2\) and \(\mathbf{a}_3\) are defined as follows:

FCC unit vectors

The discrete position \(\mathbf{r}\) of a node is defined by a triplet of integer \((i,j,k)\) such that:

\[ \mathbf{r} = i \mathbf{a}_1 + j \mathbf{a}_2 + k \mathbf{a}_3 \]

Following an idea similar to forbidden reflections used in crystallography, a triplet \((i,j,k)\) is admissible if and only if \( i+j+k=0\bmod{2} \).
