Release notes of the 3.4.4 version of TFEL, MFront and MTest

Thomas Helfer

21/09/2021

  • 1 New feature of mtest
    • 1.1 Imposed inner radius loading in pipe modelling
  • 2 New features of mfront-query
    • 2.1 Better --generated-sources query
    • 2.2 Retrieve the sources associated with specific targets
  • 3 Issues solved
    • 3.1 Issue #49: [mfront-query] query sources related to specific targets
    • 3.2 Issue #40: [mfront-query] return all the generated sources (not sorted by library)

The page describes the new functionalities of Version 3.4.4 of the TFEL project.

1 New feature of mtest

1.1 Imposed inner radius loading in pipe modelling

PTest now supports imposing the evolution of the inner radius of the pipe.

1.1.1 Example of usage

The following code shows how to impose the evolution of the inner radius of the pipe:

@RadialLoading 'ImposedInnerRadius';
@InnerRadiusEvolution{0 : 4.2e-3, 1 : 4.3e-3};

2 New features of mfront-query

2.1 Better --generated-sources query

The --generated-sources query can now have an option:

  • sorted-by-libraries (default): which returns the sources to be generated associated with each library
  • unsorted: which returns all the sources to be generated

This option may be used in conjunction with the --nomelt option.

2.2 Retrieve the sources associated with specific targets

mfront-query now have two new queries:

  • --specific-target-generated-sources which allows to retrieve the sources generated by a specific target
  • --all-specific-targets-generated-source, which displays all the sources associated with specific targets

By default, all sources associated with libraries dependencies of the specific targets are also displayed, unless the --nomelt command line argument is used.

3 Issues solved

3.1 Issue #49: [mfront-query] query sources related to specific targets

See Section 2.2 for a description of the --specific-target-generated-sources and --all-specific-targets-generated-source queries.

For more details, see https://github.com/thelfer/tfel/issues/40.

3.2 Issue #40: [mfront-query] return all the generated sources (not sorted by library)

See Section 2.1 for a description of the new features of the --generated-sources query.

For more details, see https://github.com/thelfer/tfel/issues/40.
