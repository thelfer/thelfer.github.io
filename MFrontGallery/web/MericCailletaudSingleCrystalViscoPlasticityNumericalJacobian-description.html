Parameters

  • numerical_jacobian_epsilon:
    • variable type: NumericType
    • description: perturbation value used to compute a finite difference approximation of the jacobian
    • default value for Tridimensional: 1e-07
  • epsilon:
    • variable type: NumericType
    • description: value used to stop the iteration of the implicit algorithm
    • default value for Tridimensional: 1e-14
  • theta:
    • variable type: NumericType
    • description: theta value used by the implicit scheme
    • default value for Tridimensional: 1
  • YoungModulus1:
    • variable name: young1
    • variable type: stress
    • description: le module d’Young d’un matériau orthotrope suivant la première direction d’orthotropie
    • default value for Tridimensional: 208000
  • YoungModulus2:
    • variable name: young2
    • variable type: stress
    • description: le module d’Young d’un matériau orthotrope suivant la deuxième direction d’orthotropie
    • default value for Tridimensional: 208000
  • YoungModulus3:
    • variable name: young3
    • variable type: stress
    • description: le module d’Young d’un matériau orthotrope suivant la troisième direction d’orthotropie
    • default value for Tridimensional: 208000
  • PoissonRatio12:
    • variable name: nu12
    • variable type: real
    • description: le coefficient de Poisson d’un matériau orthotrope relatif aux première et deuxième directions d’orthotropie
    • default value for Tridimensional: 0.3
  • PoissonRatio23:
    • variable name: nu23
    • variable type: real
    • description: le coefficient de Poisson d’un matériau orthotrope relatif aux deuxième et troisième directions d’orthotropie
    • default value for Tridimensional: 0.3
  • PoissonRatio13:
    • variable name: nu13
    • variable type: real
    • description: le coefficient de Poisson d’un matériau orthotrope relatif aux première et troisième directions d’orthotropie
    • default value for Tridimensional: 0.3
  • ShearModulus12:
    • variable name: mu12
    • variable type: stress
    • description: le module de cisaillement d’un matériau orthotrope relatif aux première et deuxième directions d’orthotropie
    • default value for Tridimensional: 80000
  • ShearModulus23:
    • variable name: mu23
    • variable type: stress
    • description: le module de cisaillement d’un matériau orthotrope relatif aux deuxième et troisième directions d’orthotropie
    • default value for Tridimensional: 80000
  • ShearModulus13:
    • variable name: mu13
    • variable type: stress
    • description: le module de cisaillement d’un matériau orthotrope relatif aux première et troisième directions d’orthotropie
    • default value for Tridimensional: 80000
  • RelativeValueForTheEquivalentStressLowerBoundDefinition:
    • variable name: relative_value_for_the_equivalent_stress_lower_bound
    • variable type: real
    • description: Relative value used to define a lower bound for the equivalent stress. For isotropic parameters, this lower bound will be equal to this value multiplied by the Young modulus. For orthotropic materials, this lower bound will be this value multiplied by the first component of the stiffness tensor.
    • default value for Tridimensional: 1e-12
  • n:
    • variable type: real
    • default value for Tridimensional: 10
  • K:
    • variable type: real
    • default value for Tridimensional: 25
  • tau0:
    • variable type: real
    • default value for Tridimensional: 66.62
  • Q:
    • variable type: real
    • default value for Tridimensional: 11.43
  • b:
    • variable type: real
    • default value for Tridimensional: 2.1
  • d:
    • variable type: real
    • default value for Tridimensional: 494
  • C:
    • variable type: real
    • default value for Tridimensional: 14363
  • minimal_time_step_scaling_factor:
    • variable type: time
    • description: minimal value for the time step scaling factor
    • default value for Tridimensional: 0.1
  • maximal_time_step_scaling_factor:
    • variable type: time
    • description: maximal value for the time step scaling factor
    • default value for Tridimensional: 1.79769e+308
  • iterMax:
    • variable type: ushort
    • description: maximum number of iterations allowed
    • default value for Tridimensional: 100

    • variable type: StiffnessTensor&
    • description: stiffness tensor computed from elastic material properties
