Detailed Description

HCP crystallography handler using the Wigner-Seitz cell.

HCP handles hexagonal closed-packed structures using the Wigner-Seitz cell and the standard 4 indices notation.

The unit vectors \(\mathbf{a}_1\), \(\mathbf{a}_2\), \(\mathbf{a}_3\) and \(\mathbf{a}_4\) are defined as follows:

HCP_4i unit vectors

The discrete position \(\mathbf{r}\) of a node is defined by a quadruplet of integer \((i,j,k,l)\) such that:

\[ \mathbf{r} = i \mathbf{a}_1 + j \mathbf{a}_2 + k \mathbf{a}_3 + l \mathbf{a}_4 \]
