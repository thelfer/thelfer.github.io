◆ StiffnessUpdatingPolicy

enum struct mtest::StiffnessUpdatingPolicy
strong

possible algorithms used for global convergence to update the stiffness matrix

Enumerator
CONSTANTSTIFFNESS 

the initial prediction matrix or the initial tangent coherent matrix or the initial secant matrix (normally equal to the elastic stiffness) is used for all iterations

CONSTANTSTIFFNESSBYPERIOD 

the prediction matrix or the first tangent coherent matrix or the first secant matrix is used for all iterations and updated and factorized at at periods

UPDATEDSTIFFNESSMATRIX 

the tangent coherent matrix or the secant matrix is computed and factorized at each iteration

UNSPECIFIEDSTIFFNESSUPDATINGPOLICY 

default value

◆ write()

MTEST_VISIBILITY_EXPORT void mtest::write ( const TestDescription &  ,
const std::string &   
)

write an MTest test to a file.

The file format is deduced from the extension of the file.

Currently, only extensions associated with the madnex file format (https://github.com/thelfer/madnex) are supported if TFEL is compiled with support of this file format. Those extensions are: mdnx, madnex (deprecated) or edf (experimental data file, deprecated). Note that the behaviour member of the metadata must be specified for export in the madnex file format.

[in]dMTest file description
[in]foutput file path
