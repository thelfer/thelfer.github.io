Detailed Description

This structure describes a mechanical behaviour

This class handles the fact that the mechanical behaviour data may depend on the modelling hypothesis considered, altough most of the time they won't.

This class thus stores a default mechanical behaviour data which is intendend to be valid for most hypotheses, and specific mechanical behaviour datas when required.

The term "probably" refers to the fact that it may happens that the increment is only used to get the value of the external state variable at a given date.
