Release notes of the 5.0.2 version of TFEL, MFront and MTest

Thomas Helfer

2025

  • 1 Issues fixed
    • 1.1 Issue 867: Compilation fails with intel 2022
    • 1.2 Issue 854: [tfel-math] tfel::math::fpclassify is not usable in constexpr’s context
    • 1.3 Issue 843: [python bindings] fix call to os.add_dll_directory when TFELHOME is defined
    • 1.4 Issue 798: [tfel-math] fix declaration of the data function in the View class

The page describes the new functionalities of Version 5.0.2 of the TFEL project.

This version was released on December 10, 2025 along with:

  • Version 3.0.16,
  • Version 3.1.16,
  • Version 3.2.13,
  • Version 3.3.8,
  • Version 3.4.9,
  • Version 4.0.6,
  • Version 4.1.5,
  • Version 4.2.4.

and inherits from all fixes from those releases.

1 Issues fixed

1.1 Issue 867: Compilation fails with intel 2022

For more details, see https://github.com/thelfer/tfel/issues/867

1.2 Issue 854: [tfel-math] tfel::math::fpclassify is not usable in constexpr’s context

For more details, see https://github.com/thelfer/tfel/issues/854

1.3 Issue 843: [python bindings] fix call to os.add_dll_directory when TFELHOME is defined

For more details, see https://github.com/thelfer/tfel/issues/843

1.4 Issue 798: [tfel-math] fix declaration of the data function in the View class

For more details, see https://github.com/thelfer/tfel/issues/798
