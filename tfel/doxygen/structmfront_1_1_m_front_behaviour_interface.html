write the behaviour constructor header associated with the interface. This method allow the interface to add a new constructor containing specific arguments and the call specific constructors for the behaviour data and integration data, as build by the writeBehaviourDataConstructor and the writeIntegrationDataConstructor methods respectively. This body of this constructor will be populated by the writeBehaviourConstructorBody method.
