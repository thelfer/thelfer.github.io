Detailed Description

the namespace aster contains material related to the MFront interface to the Code-Aster finite element solver.

Typedef Documentation

◆ StressFreeExpansionHandler

typedef void(* aster::StressFreeExpansionHandler) (AsterReal *const, AsterReal *const, const AsterReal *const, const AsterReal *const, const AsterInt)

prototype of a function handling stress-free expansion at the beginning of the time step

  • first paramater : driving variable
  • second paramater : driving variable at the end of the time step or driving variable increment
  • third paramater : expansion value at the beginning of the time step
  • fourth paramater : expansion value at the end of the time step
  • third parameter : spatial dimension

list of the finite strain formulation available.

Enumerator
SIMO_MIEHE 

Formulation introduced by Simo and Miehe (the default)

GROT_GDEP 

Total lagrangian formulation.

UNDEFINEDFINITESTRAINFORMULATION 

entry used when the finite strain formulation is meaningless (e.g. small strain behaviour).
